What's inside:

  • Music engine

  • Sound effect pointers and data
  • Song pointers and data
  • Note length tables (tempos)

You would think you could skip processing, since if the game is paused, nothing should be setting new music or whatever.

You would be correct, except for the suicide code! That sets MusicQueue2.

If not for processing it, the music would not change (or stop) when you used the code. Welp!
